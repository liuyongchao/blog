# 保存自动格式化符合ESLint规范
* 1.VSCode下载安装ESLint插件
* 2.下载安装Vetur
* 3.配置user.settings文件
```bash
{
  "eslint.autoFixOnSave": true,
  "eslint.validate": [
    "javascript",
    {
      "language": "vue",
      "autoFix": true
    },
    "html",
    "vue"
  ],
  "editor.quickSuggestions": {
    "strings": true
  }
}
````
